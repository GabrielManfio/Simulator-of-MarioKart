# 🏁 Mario Kart Race Simulator

Simulador de corridas inspirado no universo Mario Kart, desenvolvido em JavaScript durante meus estudos de Node.js.

O projeto simula uma corrida entre personagens com atributos próprios, utilizando elementos aleatórios e regras de jogo para determinar o vencedor da disputa.

## 🚀 Objetivo

Praticar conceitos fundamentais de programação através da criação de uma simulação baseada em mecânicas presentes nos jogos da franquia Mario Kart.

## 🛠️ Tecnologias Utilizadas

* JavaScript
* Node.js

## 📚 Conceitos Praticados

* Programação Orientada a Objetos (POO)
* Funções assíncronas (async/await)
* Modularização de código
* Manipulação de objetos
* Operadores matemáticos
* Estruturação de regras de negócio
* Simulações com eventos aleatórios

## ⚙️ Mecânicas da Corrida

O simulador executa uma corrida composta por múltiplas rodadas, onde cada personagem possui atributos específicos:

* 🏎️ Velocidade
* 🔄 Manobrabilidade
* 💪 Poder

Durante cada rodada são sorteados diferentes tipos de desafios:

* RETA
* CURVA
* CONFRONTO

Os resultados são definidos pela combinação entre os atributos dos personagens e valores aleatórios gerados durante a corrida.

Ao final, o sistema calcula a pontuação e determina o vencedor.

## 🎮 Personagens

O projeto utiliza personagens inspirados no universo Mario Kart, cada um com características próprias que influenciam diretamente o resultado da corrida.

## ▶️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/GabrielManfio/Simulator-of-MarioKart.git
```

Acesse a pasta do projeto:

```bash
cd Simulator-of-MarioKart
```

Instale as dependências:

```bash
npm install
```

Execute o simulador:

```bash
node src/index.js
```
